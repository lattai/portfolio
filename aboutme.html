Why would you use a dictionary vs a list? What are the similarities and differences between the two?

Dictionarys are used for key:value sets to be stored. Lists are items with the just one set of values. Dictionaries are in no specified order, so when printed, the output is randomized. Lists keep the order they are entered in. Both are used to store data, and can be used to create dataframes.
